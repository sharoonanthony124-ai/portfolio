├── css/

## ✏️ How to Customize

1. **Your name & info** — edit `index.html` directly
2. **Email address** — search for `sharoon.anthony@email.com` and replace
3. **LinkedIn URL** — search for `linkedin.com/in/sharoon-anthony` and replace
4. **Colors** — open `css/style.css`, edit the CSS variables at the top (`:root {}`)
5. **Contact form** — by default it simulates a send. To make it real:
   - Sign up at [Formspree.io](https://formspree.io) (free)
   - Replace the `form.addEventListener` logic in `js/main.js` with Formspree's fetch call

## 🌐 Deploy to GitHub Pages (5 minutes)

1. Create a new GitHub repo named `portfolio`
2. Upload all files (keep the folder structure)
3. Go to **Settings → Pages**
4. Set Source to `main` branch, folder `/root`
5. Hit Save — your site goes live at `https://yourusername.github.io/portfolio`

## 🎨 Tech Stack

- Pure HTML5, CSS3, JavaScript (no frameworks, no dependencies)
- Fonts: [Syne](https://fonts.google.com/specimen/Syne) + [DM Mono](https://fonts.google.com/specimen/DM+Mono) via Google Fonts
- Scroll animations via Intersection Observer API
